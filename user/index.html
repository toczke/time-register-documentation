Get time entries report

Retrieves a report of time entries for the authenticated user based on various filters.

Authorizations:
bearerAuth
query Parameters
startDate
string <date>

The start date for the date range filter.

endDate
string <date>

The end date for the date range filter.

year
integer

The year for the year and week or year and month filter.

week
integer

The week number for the year and week filter.

month
integer

The month number for the year and month filter.

detailed
boolean

If true, the response will be detailed day-by-day. If false, it will be aggregated.

Responses

/api/user/time-entries/report
